# Copywriting Skill

This skill details copywriting guidelines to produce content that ranks highly on search engines while maximizing conversion rates for local service customers.

## Copywriting Framework

1. **Avoid Keyword Stuffing:**
   - Write naturally for human users first.
   - Place target keyword phrases in primary spots (H1, introductory paragraph, H2, and title). Do not repeat keywords repetitively or list cities unnaturally.

2. **Match Search Intent:**
   - Commercial Intent (e.g. `Tree removal cost`): Provide transparent pricing guidelines, FAQs, and easy quote form fields.
   - Transactional Intent (e.g. `Emergency tree service`): Put call buttons and phone numbers front-and-center.
   - Informational Intent (e.g. `How to trim oak trees`): Provide arborist-written, structured explanations.

3. **Incorporate Local Signals:**
   - Mention local landmarks, county names, and neighboring towns to establish geo-relevance (e.g., Bingham County, Power County, Bannock County, Snake River Valley).

4. **Formatting for Readability:**
   - Use short paragraphs (2-3 sentences max).
   - Use bullet points and lists to group information.
   - Use bold text for key benefits.
